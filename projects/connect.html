Connect

Description

Connect is an innovative web application designed to revolutionize the job searching process. It allows users to create profiles, attach their skills, and post job listings demanding specific skills. The platform leverages a machine learning Word2Vec model to analyze the skills listed in profiles and job postings, generating matches and compatibility scores. Recruiters and job seekers can then review the matches and decide whether to accept or decline. In the case of mutual interest, they can utilize the built-in chat feature to arrange formal interviews. Connect offers a seamless and intuitive interface, making job searching efficient and enjoyable.

Skills Utilized

  • Python: Used to develop the backend and apply the machine learning Word2Vec model for skill analysis.
  • MongoDB: Integrated MongoDB for efficient storage and management of data.
  • HTML/CSS: Utilized to design and structure user interfaces, providing an intuitive user experience.
  • JavaScript: Implemented JavaScript for dynamic client-side interactions and real-time updates.
  • Gensim (Word2Vec Model): Employed Gensim library for machine learning techniques to analyze skills and generate matches between profiles and job postings.
