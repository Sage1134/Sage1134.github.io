Cursor Arena

Description

Cursor Arena is an engaging multiplayer game. The game features a canvas where players can draw walls using their cursors. These walls act as barriers that prevent other cursors from passing through them.

The objective of Cursor Arena is to strategically navigate through the game space while avoiding being trapped by other players' walls. Players aim to enclose other cursors within their drawn walls to prevent them from moving, ultimately striving to be the last cursor standing.

Players can move through their own walls, adding a layer of complexity and strategy to gameplay.

Skills Utilized

  • HTML: Used to create the structure and layout of the game interface.
  • CSS: Implemented CSS for styling elements and enhancing the visual appeal of Cursor Arena.
  • JavaScript: Leveraged JavaScript to implement dynamic interactions, handle requests to the backend, and facilitate real-time communication between players.
  • Python Sockets: Utilized Python sockets to establish communication channels between the game client and server, enabling multiplayer functionality. Game functionality and logic is processed in the backend to prevent players from cheating.
