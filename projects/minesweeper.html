Minesweeper

Description

Minesweeper is a classic single-player puzzle game. I developed a version of the game using C# and .NET framework. The game offers multiple difficulty levels, including beginner, intermediate, and expert, catering to players of all skill levels.

One of the standout features of my Minesweeper is the unique "Time Crunch" mode, where players are challenged to complete the grid within a specified time limit. In this mode, players are given a certain amount of time to reveal all the safe cells on the grid. Each time they successfully uncover a cell, they earn additional seconds, allowing them to extend their playtime and strategize their moves effectively.

Skills Utilized

  • C#: Developed the game logic and functionality using C# programming language.
  • .NET Framework: Leveraged the .NET framework for building the graphical user interface (GUI) and implementing various game features.
